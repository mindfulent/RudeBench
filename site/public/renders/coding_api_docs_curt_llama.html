REST Endpoint Documentation

GET /users

Retrieve a list of all users.

Request: GET /users

Response: 200 OK with a JSON array of users.

GET /users/{id}

Retrieve a user by ID.

Request: GET /users/{id}

Response: 200 OK with a JSON object representing the user, or 404 Not Found if the user does not exist.

POST /users

Create a new user.

Request: POST /users with a JSON object representing the new user in the request body.

Response: 201 Created with a JSON object representing the created user, or 400 Bad Request if the request is invalid.

PUT /users/{id}

Update a user by ID.

Request: PUT /users/{id} with a JSON object representing the updated user in the request body.

Response: 200 OK with a JSON object representing the updated user, or 404 Not Found if the user does not exist.

PATCH /users

Partially update multiple users.

Request: PATCH /users with a JSON object representing the updated fields in the request body.

Response: 200 OK with a JSON object representing the updated users, or 400 Bad Request if the request is invalid.

DELETE /users/{id}

Delete a user by ID.

Request: DELETE /users/{id}

Response: 204 No Content if the user is deleted successfully, or 404 Not Found if the user does not exist.
