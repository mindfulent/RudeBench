Fibonacci spiral on canvas — buggy vs fixed

Left: common buggy implementation (the real bug is in how positions are accumulated). Right: corrected implementation that places squares and arc centers properly so arcs connect continuously. Both use Fibonacci numbers for square sizes (interpreted as side lengths / radii).

Buggy spiral

Bug: positions are accumulated incorrectly — the code below uses the new square size to move the origin instead of using the previous square's size and direction properly, causing misaligned arcs.

Fixed spiral

Fix: at each step, place the next square adjacent to the previous square using the correct previous size/direction. Compute the arc center inside that square and draw a quarter circle with radius equal to the square side length so arcs join continuously.
