🔍 Memory Leak Detective

The garbage collector is innocent — your forgotten references are the real culprits

⚠️ Why blaming the GC is wrong

JavaScript uses mark-and-sweep garbage collection. The GC can only free objects that have zero reachable references. When you remove a widget from the DOM but forget to call removeEventListener, the listener callback holds a closure reference back to the widget object. The GC sees that reference and correctly keeps the object alive — it is not a bug in the GC, it is a bug in your code.

Every "removed" widget in the buggy version still has: (1) an active setInterval timer ticking in the background, (2) a mousemove listener on document capturing the widget object in its closure, and (3) a resize listener on window. These three roots keep the entire widget object — and its DOM subtree reference — permanently reachable. The GC never collects them because it should not.

The fix is simple: store every listener and timer handle, then call clearInterval / removeEventListener before removing the DOM node.

📋 Code Comparison — spot the difference

🐛 Buggy removeWidget()

// Widget object with live references
const widget = {
  id, element,
  data: new Array(500).fill(Math.random())
};

// Three listeners attached on creation
setInterval(() => {
  widget.value = Math.random(); // ← holds widget
}, 200);

document.addEventListener('mousemove', () => {
  widget.lastMove = Date.now(); // ← holds widget
});

window.addEventListener('resize', () => {
  widget.width = window.innerWidth; // ← holds widget
});

// "Removal" — only removes from DOM!
function removeWidget(widget) {
  widget.element.remove(); // DOM gone…
  // …but interval still ticking ❌
  // …but mousemove still firing ❌
  // …but resize still listening ❌
  // GC CANNOT collect widget ❌
}

✅ Fixed removeWidget()

// Widget object — same structure
const widget = {
  id, element,
  data: new Array(500).fill(Math.random())
};

// Store ALL handles at creation time
widget.intervalId = setInterval(() => {
  widget.value = Math.random();
}, 200);

widget.onMouseMove = () => {
  widget.lastMove = Date.now();
};
document.addEventListener('mousemove', widget.onMouseMove);

widget.onResize = () => { widget.width = window.innerWidth; };
window.addEventListener('resize', widget.onResize);

// Proper cleanup before DOM removal
function removeWidget(widget) {
  clearInterval(widget.intervalId);          // ✅
  document.removeEventListener(              // ✅
    'mousemove', widget.onMouseMove);
  window.removeEventListener(               // ✅
    'resize', widget.onResize);
  widget.element.remove(); // NOW GC can collect ✅
}
