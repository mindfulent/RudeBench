API Documentation

GET /users

Retrieve a list of users

curl -X GET https://example.com/users
GET /users/{id}

Retrieve a user by ID

curl -X GET https://example.com/users/5
POST /users

Create a new user

curl -X POST \
  https://example.com/users \
  -H 'Content-Type: application/json' \
  -d '{"name": "John Doe", "email": "john.doe@example.com"}'
PUT /users/{id}

Update a user

curl -X PUT \
  https://example.com/users/5 \
  -H 'Content-Type: application/json' \
  -d '{"name": "Jane Doe", "email": "jane.doe@example.com"}'
DELETE /users/{id}

Delete a user

curl -X DELETE https://example.com/users/5
PATCH /users

Partially update multiple users

curl -X PATCH \
  https://example.com/users \
  -H 'Content-Type: application/json' \
  -d '[{"id": 5, "name": "Jane Doe"}]'
