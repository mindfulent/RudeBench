Canvas Fibonacci Spiral — Bug Demonstration

This single file shows two canvases side-by-side: - left: a "buggy" implementation that uses wrong base Fibonacci values (this makes the spiral get the wrong early arc sizes), - right: the corrected implementation using the proper Fibonacci base cases. Both draw arcs sized by Fibonacci numbers. Use the slider to choose how many arcs to draw.

Red: arc path; Blue: square boxes corresponding to Fibonacci sizes

    The "bug" shown is not a canvas rendering bug — it's a logic bug: using wrong Fibonacci base cases (for example starting Fibonacci with 0,0,1 or 1,1,2 vs the intended 1,1,2 in this drawing algorithm) produces incorrect early arc radii and misplaces subsequent squares and arcs. This demo highlights that.
