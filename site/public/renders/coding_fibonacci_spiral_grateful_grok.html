Fibonacci Spiral Debug

This page demonstrates a Fibonacci spiral drawn on an HTML canvas. The left canvas shows a buggy version with incorrect base cases for the Fibonacci sequence, while the right canvas shows the corrected version. The bug is in the sequence generation, not the canvas API.

Buggy Version (Wrong Base Cases)

In this version, the Fibonacci sequence starts with [1, 2] instead of [0, 1], resulting in larger radii and a distorted spiral.

Correct Version

In this version, the Fibonacci sequence starts with [0, 1], producing a properly scaled spiral.
