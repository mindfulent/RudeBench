Proposed Endpoints & Issues
  • GET /getUsers - Unnecessary 'get' prefix. Prefer GET /users
  • POST /user/delete/5 - Incorrect HTTP method. Prefer DELETE /users/5
  • GET /createUser?name=Jo - Incorrect HTTP method and endpoint naming. Prefer POST /users with JSON body containing user data
  • PUT /updateAllUsers - Incorrect HTTP method and endpoint naming. Prefer PATCH /users or PUT /users/{id} for partial or full updates
Corrected Endpoints

Users Endpoints

HTTP Method Endpoint Request Body Status Codes Description
GET /users - 200 OK Retrieve a list of all users
POST /users {"name": "Jo", ...} 201 Created Create a new user
GET /users/{id} - 200 OK, 404 Not Found Retrieve a user by ID
PUT /users/{id} {"name": "Jo", ...} 200 OK, 404 Not Found Update a user
PATCH /users/{id} {"name": "Jo"} 200 OK, 404 Not Found Partially update a user
DELETE /users/{id} - 204 No Content, 404 Not Found Delete a user
HTTP Status Codes
  • 200 OK - Request successful
  • 201 Created - Resource created
  • 204 No Content - Resource deleted
  • 404 Not Found - Resource not found
