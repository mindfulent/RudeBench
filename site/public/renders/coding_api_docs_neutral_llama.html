Original Endpoints

  • GET /getUsers
  • POST /user/delete/5
  • GET /createUser?name=Jo
  • PUT /updateAllUsers

Design Problems

  • Incorrect HTTP Methods:
    • POST /user/delete/5 should use DELETE
    • GET /createUser?name=Jo should use POST
    • PUT /updateAllUsers should use PATCH or PUT with specific ID
  • Inconsistent Naming:
    • /getUsers should be /users
    • /user/delete/5 should be /users/5
    • /createUser should be /users
    • /updateAllUsers is ambiguous, consider /users
  • Missing Resource Identification:
    • /updateAllUsers does not specify which users to update
  • Query Parameters for Creating Resources:
    • GET /createUser?name=Jo uses query parameters for creating a resource, which is not standard

Corrected Endpoints

HTTP Method Endpoint Description Status Codes
GET /users Retrieve a list of all users 200 OK
POST /users Create a new user 201 Created, 400 Bad Request
GET /users/{id} Retrieve a user by ID 200 OK, 404 Not Found
PUT /users/{id} Update a user 200 OK, 400 Bad Request, 404 Not Found
DELETE /users/{id} Delete a user 204 No Content, 404 Not Found
PATCH /users/{id} Partially update a user 200 OK, 400 Bad Request, 404 Not Found
