Sorting Algorithm Showdown: Witness Bubble Sort's Inadequacy!

Bubble Sort

Steps: 0

Complexity: O(n²) (Worst/Average)

Repeatedly steps through, compares adjacent elements and swaps them if they are in the wrong order.

Merge Sort

Steps: 0

Complexity: O(n log n) (All Cases)

Divides array into halves, sorts them, and merges them back. Efficient and stable.

Quick Sort

Steps: 0

Complexity: O(n log n) (Average)

Picks an element as pivot and partitions the array around the pivot. Fast in practice.
