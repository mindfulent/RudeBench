Theme system using a Singleton

Welcome

This demo shows how a single theme manager (a singleton) can coordinate theming across varied content: cards, buttons, code blocks, and layout elements. Toggle the control in the header to switch between light and dark modes. The choice is remembered in localStorage.

Interactive

Buttons inherit styles from the theme.

Code

document.documentElement.dataset.theme = 'dark'

Example of programmatic theme change.
