Fibonacci Spiral Debugger

This page demonstrates a common "bug" when drawing a Fibonacci spiral, which is typically not a bug in the HTML Canvas API itself, but rather a misunderstanding of the Fibonacci sequence's base cases or its application to the drawing logic. The "Buggy Version" illustrates a common mistake, while the "Corrected Version" shows the proper spiral.

The Fibonacci sequence is commonly defined as F(0)=0, F(1)=1, and F(n) = F(n-1) + F(n-2) for n > 1. However, sometimes F(1)=1, F(2)=1 is used, or the sequence is started with [1,1]. The way these numbers are used as radii for the arcs, and how the center of each arc is calculated based on the previous one, is crucial.

Buggy Version (Incorrect Fibonacci Base Cases/Application)

This version might use an incorrect starting sequence (e.g., `[1, 1]` for the first two numbers directly as radii, or miscalculates arc centers based on these). This often leads to a spiral that doesn't "fit" correctly, or appears to be misaligned, especially in its initial segments.

Corrected Version (Proper Fibonacci Sequence and Drawing Logic)

This version correctly implements the Fibonacci sequence (e.g., starting with 0, 1, 1, 2, 3...) and applies these numbers as radii for quarter-circle arcs, carefully adjusting the center point for each subsequent arc to create the continuous, aesthetically pleasing spiral.
